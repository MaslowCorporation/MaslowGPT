
# set-api-key

This command allows you to set your API key in MaslowGPT. This key unlocks the powers of OpenAI ;-)

Let's go !

Run the 

```
npx maslow
``` 

command

  <img src="assets\4ab4115e49018bb848b5a3c4e3739077.png" alt="">
</p>

Choose ```set-api-key```

</p>

You can also run 


directly
